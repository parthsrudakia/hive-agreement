Hive Agreement API

Generate Hive sublease agreement PDFs programmatically. Public endpoint — no authentication required.

← Back to web interface

Endpoint

POST https://zjcmgbhxfewkbyocxodi.supabase.co/functions/v1/generate-agreement

A GET on the same URL returns a JSON description of the API.

Rate Limit

10 requests per minute per IP address. Requests beyond the limit return HTTP 429 Too Many Requests with a Retry-After header indicating seconds to wait.

Note: Rate limiting is in-memory per server instance, so the limit is approximate under heavy distributed load.

Request Body

Send Content-Type: application/json.

FieldTypeDescription
tenantName requiredstringFull name of the sublessee/tenant.
sublessorName requiredstringFull name of the sublessor.
propertyAddress requiredstringFull street address of the property.
rent requiredstringMonthly rent amount, e.g. "1650".
proRateRent optionalstringProrated first-month rent, if applicable.
securityDeposit requiredstringSecurity deposit amount.
leaseStartDate requiredstringISO date, e.g. "2025-05-01".
leaseEndDate requiredstringISO date.
agreementDate requiredstringISO date the agreement is signed.
includeLetterhead optionalbooleanDefault true. Whether to add the Hive letterhead.
format optionalstring"pdf" (default, returns binary PDF) or "base64" (returns JSON with base64-encoded PDF).

Responses

Success — format = "pdf" (default)

Returns the PDF binary with Content-Type: application/pdf and Content-Disposition: attachment.

Success — format = "base64"

{
  "filename": "Praveen Kumar Anwla Sublease Agreement.pdf",
  "mimeType": "application/pdf",
  "base64": "JVBERi0xLjMKJb..."
}

Error responses

StatusMeaning
400Invalid JSON or validation error (see details field)
405Method not allowed (use POST)
429Rate limit exceeded
500PDF generation failed

Examples

cURL — download PDF

curl -X POST https://zjcmgbhxfewkbyocxodi.supabase.co/functions/v1/generate-agreement \
  -H "Content-Type: application/json" \
  -o agreement.pdf \
  -d '{
    "tenantName": "Praveen Kumar Anwla",
    "sublessorName": "Vineet Dutta",
    "propertyAddress": "161 Van Wagenen Ave, Jersey City, NJ 07306",
    "rent": "1650",
    "securityDeposit": "1650",
    "leaseStartDate": "2025-05-01",
    "leaseEndDate": "2026-04-30",
    "agreementDate": "2025-04-25",
    "includeLetterhead": true
  }'

JavaScript — fetch and download

const res = await fetch(
  "https://zjcmgbhxfewkbyocxodi.supabase.co/functions/v1/generate-agreement",
  {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      tenantName: "Praveen Kumar Anwla",
      sublessorName: "Vineet Dutta",
      propertyAddress: "161 Van Wagenen Ave, Jersey City, NJ 07306",
      rent: "1650",
      securityDeposit: "1650",
      leaseStartDate: "2025-05-01",
      leaseEndDate: "2026-04-30",
      agreementDate: "2025-04-25"
    })
  }
);
const blob = await res.blob();
const url = URL.createObjectURL(blob);
window.open(url);

Python

import requests

r = requests.post(
    "https://zjcmgbhxfewkbyocxodi.supabase.co/functions/v1/generate-agreement",
    json={
        "tenantName": "Praveen Kumar Anwla",
        "sublessorName": "Vineet Dutta",
        "propertyAddress": "161 Van Wagenen Ave, Jersey City, NJ 07306",
        "rent": "1650",
        "securityDeposit": "1650",
        "leaseStartDate": "2025-05-01",
        "leaseEndDate": "2026-04-30",
        "agreementDate": "2025-04-25",
    },
)
with open("agreement.pdf", "wb") as f:
    f.write(r.content)
